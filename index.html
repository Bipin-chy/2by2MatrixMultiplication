Matrix multiplication

About the method

To calculate Matrix Multiplication Method you need to do following steps:

  1. The number of columns of the 1st matrix must equal the number of rows of the 2nd matrix.
  2. And the result will have the same number of rows as the 1st matrix, and the same number of columns as the 2nd matrix.
  3. The following row operations are performed on augmented matrix when required:
    • Multiply row of first matrix with the column of second matrix.
    • After multiplying add it.
    • Replace a row with the result of multiplication and addition.
